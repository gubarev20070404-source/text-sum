# Система автоматической суммаризации текстов

Веб-сервис для сокращения длинных текстов (новостей, отчётов) с сохранением основного смысла.

Учебный проект. Направление: 09.03.03 «Прикладная информатика». 2-й семестр.

---

## Возможности

- Абстрактная суммаризация на основе модели ruT5
- Примитивный метод (первые N предложений) для сравнения
- Настройка количества предложений от 1 до 10
- Загрузка текста из `.txt` файла
- История последних 5 запросов
- Обработка ошибок (пустой ввод, короткий/длинный текст)
- Автоматическое обрезание текста до 3000 символов для стабильной работы модели

## Как запустить локально

```bash
git clone https://github.com/gubarev20070404-source/text-sum.git
cd text-summarizer
pip install -r requirements.txt
python app.py
```
После запуска в консоли появится адрес http://127.0.0.1:7860 – откройте его в браузере.

**Примечание:** При первом запуске модель ruT5 скачается автоматически (около 1 ГБ). Это займёт несколько минут.
